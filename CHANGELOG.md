# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-30

### Added
- Complete initial release of WeatherFetch.
- Integrated Open-Meteo API for global weather forecasting, historical data, and geocoding.
- Integrated Windy Map embed for interactive, live WebGL weather and wind visualization.
- Dark and Light themes with dynamic CSS swapping.
- Complete JavaFX architecture separated into robust MVC patterns.
- Comprehensive UI styling incorporating premium glassmorphism and modern aesthetics.
- Background asynchronous data loading using `CompletableFuture`.
- Gradle build system and dependency management configuration.
- JUnit 5 and Mockito suite for testing core services.
